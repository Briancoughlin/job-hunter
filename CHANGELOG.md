## [1.0.0] - 2026-06-29

### Added
- **CV parser** — upload a LinkedIn PDF; Claude extracts skills, experience, seniority, domains, and industries into a structured profile
- **Multi-board job search** — searches LinkedIn, Indeed, Glassdoor, and ZipRecruiter via `python-jobspy`
- **AI scoring** — Claude scores each job 0-100 against your profile with skills match, seniority match, domain match, strengths, gaps, and a fit summary
- **Interview probability** — low / medium / high / very high rating per role
- **Ranked results** — sortable table filterable by minimum score and recommendation flag
- **Save and track** — save roles, mark as applied, export results to CSV
- **Profile persistence** — parsed profile cached locally; no need to re-upload CV each session
- **Configurable LLM gateway** — Base URL field supports internal LLM proxies alongside the standard Anthropic API
- **Windows desktop launcher** — `run.bat` + `run.vbs` open the app silently in Edge app mode with no terminal window
- **Taskbar shortcut** — `create_shortcut.ps1` generates a branded `.lnk` file that can be pinned to Start or the taskbar
- **Custom icon** — red target icon generated by `create_icons.py` using Pillow
- **CI pipeline** — ruff lint and import check on every push via GitHub Actions
- **Launcher integrity check** — CI alerts and blocks pushes to `master` if `run.bat`, `run.vbs`, or `create_shortcut.ps1` are modified
- **Dependabot** — weekly Python dependency scanning
- **Python 3.12+ compatibility** — `python-jobspy` installed with `--no-deps` to avoid numpy wheel build failures on newer Python versions
